títulos html

O <:h1> serve para definir o título pricipal da página, o <h2> serve para definir o subtítulo,o <h3> serve para definir o terceiro nível de título ou subtítulo, o <h4> serve para definir o título ou subtítulo de quarto nível, o <h5> serve para definir o título subtítulo de quinto nível, o <h6> sere para definir o título mais baixo da página
